## Currency2U Frontend

Frontend module builded using Next.js framework, that uses typescript, tailwind and React, mainly.

#### Usage

First, create a .env file in this folder and populate it with the following valid variables:

```env
API_URL=http://localhost:5000
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

Then, run at terminal, inside this folder:

```bash
pnpm install
```

And then, immediately afterwards:

```bash
pnpm run dev
```
